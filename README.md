# Employee tenure AD Visualizer

Small app that visualizes employees tenure based on Active Directory data.

## Quick start

1. Run `Export-From-AD.ps1` in Powershell to export AD data to a local json file.
2. Serve app via any HTTP server (`python -m http.server 8000`), than open in browser. Custom data file can be configured in `config.json`.
